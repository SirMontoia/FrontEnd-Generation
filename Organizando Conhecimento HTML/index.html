Resumo dos conhecimentos HTML

Linhas de código preliminares

  • DocType HTML = Indica que trabalharemos com a versão HTML mais atualizada no nosso projeto
  • HTML Lang = "" = indica a linguagem de nossa página web (padronizar para pt-br no nosso caso)
  • Head = informações interpretadas antes das outras pelo navegador
    1. charset = Qual o padrão de caracteres trabalhados (char = character set = mudar para)
    2. name e view port = Indicam como o conteúdo será visualizado (width nesse caso está definido como o padrão do dispositivo que acessa a página)
    3. title = Título da página (nome que fica na aba em cima)
Códigos Básicos HTML

Comandos Básicos

  • body = Corpo do site. Comporta os textos, imagens, vídeos, links, etc
  • H1 - H6 = Headers. Usados para títulos e sub-títulos. Apenas um "H1" pode ser usado, o resto pode ser quantos quiser
  • P = Paragraph, cria parágrafos. Usado para quando se quer fazer linhas em parágrafos diferentes
  • Img = Usado para colocar imagem. SRC é a fonte (internet ou pastas assets)
  • Iframe = Usado para pegar mapas e/ou vídeos. Clicar em "Compartilhar" e pegar os campos relacionados a "HTML"
  • OL e UL = Order List e Unorder List respectivamente. Primeiro lida lida com marcadores numéricos de separação e outro lida com marcadores geométricos (bolinhas por exemplo)
  • a href = Anchor. Comando usado para redirecionar para outra página (definida no src) quando for acessado. Se quiser abrir outra aba, inclua "target_blank" depois do link
